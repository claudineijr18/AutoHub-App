# 🚗 AutoHub

Sistema mobile desenvolvido para auxiliar no gerenciamento e acompanhamento de veículos, manutenções e informações automotivas de maneira prática e organizada.

---

## 📱 Sobre o Projeto

O **AutoHub** é um aplicativo Android criado com foco em facilitar o controle de informações relacionadas a veículos.

O sistema permite registrar e acompanhar manutenções, organizar dados importantes do automóvel e centralizar informações em um único aplicativo.

O projeto está sendo desenvolvido utilizando:

* Java
* Android Studio
* SQLite/Firebase (dependendo da implementação futura)
* XML para interface gráfica

---

## ✨ Funcionalidades

### ✅ Funcionalidades atuais

* Cadastro de veículos
* Tela inicial do aplicativo
* Navegação entre telas
* Cadastro de manutenções
* Organização de informações automotivas

### 🚧 Funcionalidades futuras

* Histórico completo de manutenção
* Controle de gastos
* Notificações de revisão
* Cadastro de abastecimentos
* Backup em nuvem
* Login de usuários
* Relatórios e estatísticas
* Modo offline

* Configure um emulador Android
  OU

## 📌 Objetivo do Projeto

O objetivo do AutoHub é fornecer uma solução prática para gerenciamento automotivo, ajudando usuários a manterem controle sobre revisões, manutenções e informações importantes do veículo.

Além disso, o projeto também possui finalidade acadêmica e de aprendizado em desenvolvimento mobile Android.
